Getting Started

This living style guide is to promote visual consistency and aim to speed up UI development times at Me2.

If you are looking for resources on style guides, check out styleguides.io. There are a ton of great articles, books, podcasts, talks, and other style guide tools!

Still a working progress! Watch this space, more stuffs will be added as we build out the platform :)
